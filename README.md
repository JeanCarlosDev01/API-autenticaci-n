# API-autenticación
Servicio de API desarrollada usando Nodejs y Express que permite registrar un usuario en la base de datos y realizar la autenticación.

* Servidor Local
    - XAMPP
    <br>
* Base de Datos
    - MySQL Workbench
    <br>
* Herramienta de pruebas
    - Postman
